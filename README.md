# Requirements
 - Java 21

# How to set the API Key
In Windows, you can do:
```
$env:ROUTELLM_API_KEY="your_key_here"
```

# How to set JAVA_HOME
In Windows, you can do:
```
$env:JAVA_HOME="Your\java\path\here"
```

# How to run
```
./gradlew bootRun
```
